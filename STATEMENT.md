## Problem Statement
Modern lifestyle pressures lead to increased stress levels among individuals, particularly students and working professionals. There is a significant need for accessible, private, and immediate stress assessment and management tools that can provide scientific evaluation and practical relief techniques without requiring professional intervention or complex setups.

## Scope of the Project
The project scope encompasses the development of a comprehensive stress management system including:
- A validated stress assessment questionnaire with scoring mechanism
- Multiple therapeutic exercise modules for stress relief
- Content management for psychological well-being tools
- Command-line interface for universal accessibility
- Modular architecture for future enhancements

## Target Users
- Students experiencing academic pressure and examination stress
- Working professionals facing workplace stress and burnout
- Individuals seeking private mental health self-assessment
- Anyone interested in preventive mental wellness practices
- Users preferring text-based applications over graphical interfaces

## High-Level Features
1. Stress Assessment Module - 5-question evaluation with scoring
2. Breathing Exercise Module - Guided deep breathing patterns
3. Affirmations Module - Curated positive mindset statements
4. Tips Module - Practical daily stress management advice
5. Gratitude Module - Interactive gratitude journaling exercise
6. Menu Navigation System - User-friendly command interface
