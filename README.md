# Stress Management Application

## Overview
A Python-based command-line application for stress assessment and management through therapeutic exercises and validated psychological tools.

## Features
- Stress Level Assessment with scoring system
- Deep Breathing Exercises for immediate relief
- Positive Affirmations for mindset shift
- Practical Stress Management Tips
- Gratitude Journaling Exercise
- Modular and extensible architecture

## Technologies/Tools Used
- Python 3.x
- Standard Python Libraries (time, sys)
- Git for version control
- VS Code as development environment

## Steps to Install & Run the Project
1. Ensure Python 3.6+ is installed on your system
2. Clone this repository: `git clone [repository-url]`
3. Navigate to project directory: `cd stress-management-app`
4. Run the application: `python main.py`
5. Follow the menu prompts to access different features

## Instructions for Testing
1. Run the main application: `python main.py`
2. Test all menu options (1-6) for functionality
3. Run validation tests: `python test_basic.py`
4. Verify input validation by entering invalid inputs
5. Check scoring accuracy with known test cases
